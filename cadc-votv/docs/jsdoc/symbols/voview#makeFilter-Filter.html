clearColumnFilters()
Clear any column value filters currently set on the VOTABLE.
doFilter()
Produces a single page's worth of VOTABLE data. After filtering is complete, the filterCallback function is called with the results.
Parameters:
{function} filterParams.filterCallback
Function to call when the result of the filtering is completed. The one argument to this function is an XML DOM object of the filtered VOTABLE. If omitted then use function specified in constructor.
setColumnFilters(filterKeys)
Set column value filters on the VOTABLE. Any filters all ready set on columns not specified in the filterKeys are retained.
Parameters:
{setFilterParams.filterKeys[]} filterKeys
An array of type columnFilterKey.
setInputTable()
Set the VO table used as input to the filter. Once in a table is specified, flags are set so that the table will be reprocessed and filtered when the doFilter method is called.
Parameters:
{XML DOM Object} setInputParams.tableDOM
The XML Dom object for the input VOTABLE.
setRowRange()
Set range of rows to be extracted from the VOTABLE.
Parameters:
{integer} setRangeParam.firstRow
The first row of the range.
{integer} setRangeParam.lastRow
The last row of the range.
setSelectRows()
Set the criteria which determines which rows will be selected when the "select all" button is activated. **ONLY string argument is currently implemented**
Parameters:
{string|function} setSelectParams.criteria
If a string, use it to match against the contents of the VOTABLE row. The contents of the VOTABLE XML row will be searched and the row will be selected if it contains the input string. If a function, then a function which will be called for each row in the XML VOTABLE, with an XML DOM object of the row as its only argument. The function should return a boolean indicating whether the row should be selected or not.
setSortColumns(sortKeys)
Set the columns to use for sorting the table, and the sorting direction for each column.
Parameters:
{setSortParams.sortKeys[]} sortKeys
An array of type sortColumnKey. The first key in the array has the highest precedence.
