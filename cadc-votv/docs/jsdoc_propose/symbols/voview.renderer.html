Method Detail
addColumn(position, formatter)
Display an additional column in the HTML table at the designated location. This column can be filled in by the application user.
Parameters:
{integer} position
Integer specifying the position of the column in the HTML table. A value of 0 indicates a column at the beginning of the table, a value of -1 indicates a column at the end of the table.
{function} formatter
A function to be called to set the value to be placed in the cells of the column. The function is called once for each row of the HTML table. The function is called with two arguments. The first argument is the HTML DOM object for the row, and the second argument is the HTML DOM object for the entire table.
formatColumn(column, format)
Add additional formatting to a column. A function can be called when formatting the column, which can be used for formatting other parts of the row as well.
Parameters:
{string|integer|Regex} column
Column to be formatted. This can be specified either as: 1) an integer (the column number in the original order); 2) a string matching a substring in the column name; 3) a regular expression matching the column name.
{string|function} format
The formatting information for the column. This can be either: 1) a string, in which case it will replace the current value of each of the cells of the column. If the string contains "@@", it will be replaced by the current cell value; 2) a function, in which case the function will be called for each row in the column. The function is called with two arguments. The first argument is the HTML DOM object for the cell, and the second argument is the HTML DOM object for the entire row.
{integers[]} getSelectedRows()
Return an array listing the rows of the table currently selected.
Returns:
{integers[]} rows An array of integers containing a list of selected row numbers. The numbers correspond to the row numbers of the original VOTABLE.
{HTML DOM Object} render(subwidget)
Generate the HTML for the entire VOView display, or for some part (sub-widget) of the display.
Parameters:
{string} subwidget
The part of the VOView display to generate. If undefined or set to "all", the entire VOView display is generated. Available sub-widget arguments are: "title", "paging", "rowSelection", "table", "columnArranging" and "Parameters".
Returns:
{HTML DOM Object} The DOM document fragment for the sub-widget.
setDisplayedColumns(columnorder)
Set the order and number of columns to display.
Parameters:
{integer[]|string[]|integer} columnorder
If an array, contains a list of columns in the order in which they are to appear. If an integer array, the numbers correspond to the original order of the columns in the VOTABLE. If a string array, the strings must correspond to column names. If a scalar integer, then this value sets the maximum number of columns to initially display in the HTML table.
setEnableRowSelection(display)
This activates VOView functionality for tracking and manipulating row selection. It displays a column at the beginning of the table with checkboxes for selecting rows.
Parameters:
{boolean} display
If true, enable row selection.
setSelectCallBack(add, del)
Set functions to be called when a row is either selected or unselected.
Parameters:
{function} add
Function to be called when a row is selected. The function is called with HTML DOM object for the row as its argument.
{function} del
Function to be called when a row is unselected. The function is called with HTML DOM object for the row as its argument.
setSelectedRows(rows)
Set rows which are selected, i.e. they are shown as selected when the table is initially displayed.
Parameters:
{integers[]|function} rows
If an array of integers, contains a list of row numbers. The numbers correspond to the row numbers of the original VOTABLE. If a function, then a function which will be called for each row in the XML VOTABLE, with an XML DOM object of the row as its only argument. The function should return a boolean indicating whether the row should be selected or not.
setTitle(titletext)
Set the title of the table display. If not set, VOView tries to determine a suitable title on its own.
Parameters:
{string} titletext Optional
Text for the title.
setUpdateCallBack(update)
Set a function to call when ever the table display is updated.
Parameters:
{function} update
Function to be called when the display is updated. The function is called with no arguments.
